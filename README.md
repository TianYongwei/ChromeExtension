# Mute Zhihu

> mute_zhihu(低调浏览知乎插件)

#### 已完成功能

- 自动识别目标网站zhihu.com
- 按钮去色  
- 问题标题去掉加粗  
- 点赞按钮去色  
- 按钮的蓝色边框  
- 隐藏知乎logo  
- 隐藏相关推荐和相关问题  
- 隐藏通知按钮，消息按钮
