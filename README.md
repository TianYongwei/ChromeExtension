### mute_zhihu(低调浏览知乎插件)
---

#### 已完成功能
- “低调化”处理  
1、按钮去色  
2、问题标题去掉加粗  
3、点赞按钮去色  
4、按钮的蓝色边框  

- 隐藏处理  
1、隐藏知乎logo  
2、隐藏相关推荐和相关问题 
3、隐藏通知按钮，消息按钮
