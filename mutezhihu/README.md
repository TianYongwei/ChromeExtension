# Mute Zhihu

## :mahjong: 已完成功能 

- [x] 自动识别目标网站zhihu.com
- [x] 按钮去色  
- [x] 问题标题去掉加粗  
- [x] 点赞按钮去色  
- [x] 按钮的蓝色边框  
- [x] 隐藏知乎logo  
- [x] 隐藏相关推荐和相关问题  
- [x] 隐藏通知按钮，消息按钮
